## Installation

Clone the repository:
```
$ git clone https://github.com/tje3d/Simple-Html-Mailbox
```

Install Dependecies Using [Bower](https://bower.io):

```
$ bower install bootstrap bootstrap-rtl fontawesome wysihtml
```
